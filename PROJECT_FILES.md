# NeuroMind AI - Project Files Overview

Complete inventory of all files created for the NeuroMind AI mental health assessment application.

## 📁 Frontend Files

### Pages
- **`app/page.tsx`** - Home page with hero section, features overview, and FAIR ethics explanation
- **`app/assess/page.tsx`** - Mental health assessment form with text input and API integration
- **`app/history/page.tsx`** - Assessment history viewer with localStorage management

### Components
- **`components/AssessmentResults.tsx`** - Results display showing risk classification, confidence, and detailed analysis

### Layout & Styling
- **`app/layout.tsx`** - Root layout with metadata and font configuration (updated)
- **`app/globals.css`** - Global styles with professional color scheme for mental health app (updated)
- **`tailwind.config.ts`** - Tailwind CSS configuration (auto-generated, no changes needed)

### Configuration
- **`.env.example`** - Example environment variables for frontend
- **`.env.local`** - Local environment file (created during setup, not in repo)

## 🔧 Backend Files

### Main Application
- **`backend/main.py`** - FastAPI application with assessment endpoint and model loading
- **`backend/generate_data.py`** - Synthetic mental health dataset generation for training
- **`backend/train_model.py`** - ML model training using scikit-learn (TF-IDF + Logistic Regression)

### Configuration & Setup
- **`backend/pyproject.toml`** - Python dependencies and project metadata
- **`backend/Procfile`** - Render deployment configuration with gunicorn + uvicorn
- **`backend/setup.sh`** - Automated setup script for dependencies and model training
- **`backend/.gitignore`** - Git ignore rules for Python virtual environment and model files
- **`backend/README.md`** - Backend-specific documentation and API details

### Generated Files (not in repo)
- **`backend/models/model.pkl`** - Trained Logistic Regression model (generated by train_model.py)
- **`backend/models/vectorizer.pkl`** - TF-IDF vectorizer (generated by train_model.py)
- **`backend/training_data.json`** - Training dataset (generated by generate_data.py)

## 🔍 Key Implementation Details

### Frontend Technologies
- **Next.js 16** - Latest React framework with App Router
- **TypeScript** - Type-safe code
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Pre-built accessible components
- **localStorage** - Client-side data persistence
- **Fetch API** - Backend communication

### Backend Technologies
- **FastAPI** - Modern Python web framework
- **scikit-learn** - Machine learning library
- **TF-IDF Vectorizer** - Text feature extraction
- **Logistic Regression** - Classification model
- **Pydantic** - Data validation
- **CORS Middleware** - Cross-origin requests

### Data Flow
```
User Browser (Frontend)
    ↓ (Form Input)
localStorage (History)
    ↓ (Fetch POST /assess)
FastAPI Backend
    ↓ (TF-IDF Vectorization)
ML Model (scikit-learn)
    ↓ (Prediction + Confidence)
JSON Response
    ↓ (Display Results)
React Components
    ↓ (Save to localStorage)
Backend Complete
```
